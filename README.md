# Archimédův zákon - Interaktivní web

Moderní, interaktivní a plně responzivní vzdělávací web o Archimédově zákonu s fyzikální simulací, kalkulačkou a kvízem.

## 💻 Lokální spuštění

Spuštění na vašem počítači je velmi jednoduché:
1. Stáhněte si zdrojové soubory.
2. Poklikejte na soubor `index.html`. Web se okamžitě otevře v prohlížeči.

---

## 🚀 Jak spustit web na GitHubu (GitHub Pages)

Chcete-li web zdarma publikovat na internetu pomocí GitHub Pages, postupujte podle těchto kroků:

1. **Vytvořte repozitář na GitHubu:**
   - Přihlaste se na [GitHub.com](https://github.com/).
   - Klikněte na tlačítko **New** (Nový repozitář).
   - Pojmenujte repozitář (např. `archimeduv-zakon`) a nastavte ho jako **Public** (Veřejný).
   - Klikněte na **Create repository**.

2. **Nahrajte soubory:**
   - Nahrajte do repozitáře všechny tři hlavní soubory přímo do hlavní větve (root složky):
     - `index.html`
     - `styles.css`
     - `script.js`

3. **Aktivujte GitHub Pages:**
   - V repozitáři přejděte do horní záložky **Settings** (Nastavení).
   - V levém menu klikněte na položku **Pages**.
   - V sekci *Build and deployment* -> *Source* vyberte **Deploy from a branch**.
   - Pod tím v rozbalovacím menu větví vyberte **main** (případně `master`) a složku **/ (root)**.
   - Klikněte na tlačítko **Save** (Uložit).

4. **Hotovo!**
   - GitHub během několika desítek sekund web spustí.
   - Adresu svého webu uvidíte na stejné stránce (Pages) nahoře v zeleném boxu. Formát adresy je:
     `https://<vaše-uživatelské-jméno>.github.io/<název-repozitáře>/`
